# AI Image Builder

AI Image Builder is a modern **MERN stack** application that allows users to generate unique and creative images using **artificial intelligence**. The platform provides a seamless experience for signing up, logging in, generating AI-powered images, and managing user-generated content. It showcases a full-stack web development project with frontend deployment on **Vercel**, backend on **Render**, and a cloud database using **MongoDB Atlas**.

- **Secure API Communication**: Backend routes protected and CORS configured for deployed frontend.
